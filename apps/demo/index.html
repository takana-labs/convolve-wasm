About

Private by architecture

Browser decoding, a dedicated module worker, and a Rust/WASM DSP core keep the complete rendering pipeline inside this tab.

Inputs and output

Use WAV or M4A supported by your browser and operating system. Every successful render is a stereo 48 kHz signed PCM24 WAV.

No hidden fallback

There is no upload service, server processor, or bundled @ffmpeg/core. Unsupported M4A codecs fail explicitly.
